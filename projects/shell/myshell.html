My Shell Program


This shell program is created using there main principles, a parser, an executer, and a line editor. The line editor is what users type into the program as commands to run. They can use the left and right arrow keys to move the cursor around to insert and delete at various places, or the up and down arrow keys to access the command history. Then users can press enter to send the input to the he parser. The parser is created using Yacc (Yet Another Compiler Compier) and Lex (Lexical Analyzer) to break down the user input into rules. The rules are then used to build a command table. The command table what it sounds like, simply a table (vector in C) that holds commands to be executed. The executer then uses regular C code to execute the commands in the command table. It deals with piping, file redirection, builtin commands, and much more.
