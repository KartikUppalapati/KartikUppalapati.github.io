My Music Downloader


I created this application to help automate the process of downloading music using YouTube. Using the open source and publicly available command line program YouTube-DL, which can be found on GitHub, I combined this application with the SpotifyAPI to gather important metadata like the Title, Artist, Album, and Album cover and then used Mp3agic to create a new .mp3 file with the proper metadata and album art. I've kept the download file as an executeable jar since this application is meant for high level users.
